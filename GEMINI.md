## Windows 中文编码规则

当前环境是 Windows，中文文件可能因为 PowerShell 默认编码导致乱码。

读取中文 txt/md/csv 文件时：
1. 优先使用 Python 读取：
   open(path, "r", encoding="utf-8-sig", errors="replace")
2. 如果使用 PowerShell，必须显式指定：
   Get-Content -Encoding UTF8 "文件路径"
3. 不要直接使用：
   Get-Content "文件路径"
4. 如果 UTF-8 读取仍然乱码，再尝试 GBK：
   open(path, "r", encoding="gbk", errors="replace")
5. 输出中文文件时统一使用 UTF-8 with BOM 或 UTF-8。
